# Contributing to ArabicSurvey

## How to Contribute

### 1. Fork the Repository

First, fork the repository on GitHub by clicking the "Fork" button at the top right of the repository page.

### 2. Clone Your Fork

Clone your fork to your local machine:

```bash
git clone https://github.com/YOUR_USERNAME/ArabicSurvey.git
cd ArabicSurvey
```

### 3. Create a New Branch

Create a new branch for your changes:

```bash
git checkout -b add-papers-branch-name
```

### 4. Add Papers to README.md


#### Formatting 

Add papers to the appropriate category in `README.md` using this format:

```markdown
- Paper Title ([Link](https://example.com/paper-url)).
```

#### Categories

Papers should be added to one of these categories:

1. General
2. Enabling Tools (Morphology, Diacritization, Grammar, etc.)
3. Text Classification & Mining
4. Named Entity Recognition
5. Question Answering
6. Sentiment Analysis
7. Speech
8. Summarization
9. Conversational AI (Chatbots & Dialogue systems)
10. Word Sense Disambiguation (WSD)
11. Dialects
12. Plagiarism Detection Systems
13. Machine Translation
14. Handwriting Recognition & OCR
15. Offensive Language Detection
16. Fake news and Spam
17. Ontology and Resources
18. Information Retrieval
19. Opinion Mining
20. Knowledge Graph and Topic Modeling
21. Query Expansion
22. Religious Text
23. LLMs
24. Others (Steganography, Transfer Learning, Deep learning, etc.)

If a paper doesn't fit any category, you can suggest a new category in your pull request.

### 5. Commit Your Changes

Stage and commit your changes:

```bash
git add README.md
git commit -m "Add [paper title] to [category]"
```

### 6. Push to Your Fork

Push your changes to your fork:

```bash
git push origin add-papers-branch-name
```
